# Product Vision

## Working Name

Market Insight Graph

## Core Idea

Build an insight-gathering engine that collects market evidence, extracts meaningful signals, links them into an interactive knowledge graph, and turns the resulting insight into strategy, content, and offer direction.

This should feel less like a dashboard and more like a living strategic notebook: structured enough to act on, flexible enough to support original thinking.

## Primary User

An independent strategist, creator, founder, consultant, or operator who wants to build sharper market understanding and turn that understanding into articulate ideas, positioning, offers, and content.

## Jobs To Be Done

- When I see useful market material, I want to capture it quickly so it does not disappear.
- When I collect research, I want the system to extract pains, desires, objections, trends, and language patterns.
- When a pattern repeats, I want to know whether it is becoming a real insight.
- When I review my insights, I want to see what evidence supports them and what contradicts them.
- When I need to write or make a strategic decision, I want to pull from clear, source-backed thinking.

## Product Principles

1. **Evidence before insight**
   Every serious insight should connect back to source material.

2. **Signals before synthesis**
   The system should preserve small observations before turning them into big claims.

3. **Human judgment stays central**
   The machine proposes links, clusters, and briefs. The user approves, edits, and curates.

4. **Connection compounds**
   The value should grow as insights link to audience pains, strategic bets, content ideas, and decisions.

5. **Actionable output**
   The system should produce usable strategy and content material, not just stored research.

## Product Modes

### Inbox Mode

Review newly captured material. Decide whether to ignore it, save it as evidence, extract a signal, attach it to an existing insight, or create a new insight.

### Graph Mode

Explore the interactive Zettelkasten-style network of evidence, signals, insights, audience segments, pains, trends, competitors, content ideas, and strategic bets.

### Brief Mode

Read structured outputs: top insights, emerging themes, contradictions, content opportunities, strategic implications, and questions to investigate next.

## Long-Term Vision

The system becomes a strategic thinking partner that can answer questions like:

- What audience pains are becoming more urgent?
- Which insights have the strongest evidence?
- Which strategic bets are under-supported?
- Which competitor messages are converging?
- Which content themes are overused but still in demand?
- What should I write about this week?
- What belief is forming across the research that I have not articulated yet?
