# Roadmap

## Phase 0: Foundation

Goal: define the product and its working structure.

Deliverables:

- product vision
- architecture outline
- data model
- source strategy
- insight brief format
- initial roadmap

## Phase 1: Manual Insight Workspace

Goal: create a usable workflow before broad automation.

Features:

- manual capture inbox
- source registry
- evidence records
- signal records
- insight brief creation
- manual links between records
- search and filters

Success criteria:

- useful research can be captured in under one minute
- evidence can be turned into signals
- signals can support insight briefs
- insights can link to each other

## Phase 2: Graph Exploration

Goal: make the knowledge base feel interactive and connected.

Features:

- graph view
- node detail panel
- relationship creation
- filters by topic, audience, source, confidence, and status
- suggested links between related nodes

Success criteria:

- user can explore how evidence, signals, and insights connect
- weak or unsupported insights are visible
- content and strategy ideas can be traced back to evidence

## Phase 3: Guided Synthesis

Goal: help turn repeated signals into clear briefs.

Features:

- signal clustering
- draft insight generation
- contradiction detection
- confidence scoring
- weekly insight review
- open question generation

Success criteria:

- weekly review produces usable strategic briefs
- the system identifies emerging themes without overwhelming the user
- every generated brief keeps evidence attached

## Phase 4: Source Automation

Goal: bring in recurring market material with controlled noise.

Features:

- RSS/feed ingestion
- competitor watchlists
- newsletter imports
- page change snapshots
- source health checks
- deduplication

Success criteria:

- high-value sources update automatically
- the inbox remains reviewable
- repeated market movement becomes easier to spot

## Phase 5: Content And Strategy Flywheel

Goal: convert insight into action and feed response back into the system.

Features:

- content idea generation
- draft outlines
- positioning notes
- offer hypotheses
- strategic bet tracking
- published content performance imports
- audience response capture

Success criteria:

- content ideas are traceable to insights
- strategic decisions are backed by market evidence
- audience response becomes new evidence
