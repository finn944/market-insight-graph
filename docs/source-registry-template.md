# Source Registry Template

Use this to define the first sources the system should care about.

```text
Name:

Type:
audience_community | competitor | newsletter | podcast | report | social_account | search_signal | owned_note | customer_conversation | content_performance

URL or Location:

Why It Matters:

Collection Method:
manual | rss | api | import | snapshot | future

Refresh Cadence:
daily | weekly | monthly | ad hoc

Priority:
low | medium | high

Signals To Watch For:
- pain points
- desires
- objections
- language patterns
- competitor moves
- trends
- questions

Notes:
```
